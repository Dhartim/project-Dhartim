Data

Data can be found in .PDF format on IUCN Summary website and was fetched on 10th April, 2020.

"This data is made available for public use and it's license and citation information can be found here"

IUCN (International Union for Conservation of Nature) 2019. The IUCN Red List of Threatened Species. Version 2020-1. http://www.iucnredlist.org. Downloaded on 10 April 2020.

For this project, merge more than two dataset, cleaned and wrangle data to create useful data for visualization.

Files use are Table 6a: Red List Category summary country totals (Animals) and Table 5: Threatened species in each country (totals by taxonomic group) which an be found here
Table 5: Threatened species in each country (totals by taxonomic group)
Column Name Description
Country: Names of Country
Mammals: Number of Mammals Threatened in each country
Birds: Number of Birds Threatened in each country
Fishes: Number of Fishes Threatened in each country
Amphibians: Number of Amphibians Threatened in each country
Reptiles: Number of Reptiles Threatened in each country
Molluscs: Number of Molluscs Threatened in each country
Plants: Number of Plants Threatened in each country
Fungi and Protist: Number of Fungi and Protist Threatened in each country
Total: Number of Total species Threatened in each country
Table 6a: Red List Category summary country totals (Animals)
Column Name Description
Country: Names of Country
EX: Number of Extinct species in each country
EW: Number of Species Extinct in Wild in each country
Subtotal: Total Number of Species Extinct in each country
CR(PE): The tags 'Possibly Extinct' have been developed to identify Critically Endangered species that are likely already extinct, but require more investigation to confirm this.
CR(PEW): The tags 'Possibly Extinct in the Wild' have been developed to identify Critically Endangered species that are likely extinct in the wild, but require more investigation to confirm this
Subtotal(EX+EW+CR(PE)+CR(PEW)): Total Number of Species Extinct or Possibly Extinct in each country.
CR: Number of Species Critically Endangered in each country
EN: Number of Species Endangered in each country
VU: Number of Species Vulnerable in each country
NT: Number of Species Near Threatened (includes LR/nt - Lower Risk/near threatened) in each country
LR/cd: Number of Species Lower Risk/conservation dependent in each country
DD: Number of Species Data deficient in each country
LC: Number of Species Least Concern (includes LR/lc - Lower Risk, least concern) in each country
Total: Total Number of Species in Red List in each country
